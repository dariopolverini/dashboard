PROJECT NAME

Al Bada Constructed Wetland

CLIENT

NEOM/Enowa

CONTRACTORS

Bauer Resources Saudi – Main Contractor

Energia – Primary Subcontractor

PROJECT DESCRIPTION

The Al-Badaa Lagoon is currently used inefficiently as a dumping site for sludge, posing significant health and safety risks. ENOWA aims to minimize these risks and transform the lagoon into a sustainable solution.

ENOWA plans to transform the lagoon into a sustainable constructed wetland (CW) to treat excess sludge, domestic wastewater, rejected tankers, and existing lagoon sludge, operating as a zero liquid discharge (ZLD) plant.

The CW will eliminate discharge by using evapotranspiration, providing an eco-friendly solution to the lagoon's legacy issues.
